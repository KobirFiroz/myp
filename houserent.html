House Rent system

We have considered this app about the habitant of Dhaka city. The main reason behind this, most of the people in this city are facing accommodation problems, such as, not getting suitable home along with finding the right location. However, the biggest problem is wasting a lot of time to getting this house. Solving those problem, we have developed this app.

Firstly, after completing the registration an owner and user can choose their specific option. If a client is owner, he can be able to add home rent, address of the location, and type of the apartment with details. This data will store in database. If a client login as user, he can see all apartment. In addition to this, by searching according to the category, then he can find the area of the house with its address. The house that he chooses, he can save it to see later.

Besides, Google map helps to finding proper location as well as user can contract with owner by the messaging system. After completing renting procedure, owner can delete, add the house so that he would not bother later. The admin maintains all this thing exceptionally database and those owners who want to add or delete. We finally believe that, by using this app peopleâ€™s hassle will be reduced a lot.

It uses Content Provider, HTML (for printing the calculation) & other basics android components such as Fragment, RecyclerView, Thrads etc.
